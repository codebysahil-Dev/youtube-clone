# YouTube Clone

HTML/CSS project by Sahil (IGNOU BCA 1st year).

## What's inside
- YouTube homepage layout
- Header with search, voice, upload, notifications
- Sidebar with navigation links
- 12 video cards in grid layout
- Responsive design (4/3/2 columns)

## Features
- YouTube homepage layout with header, sidebar & video grid
- Click on any thumbnail to open the YouTube video
- Hover effect on video cards (scale + shadow)
- No I-beam cursor on text
- Responsive design

## Live
Open `index.html` in browser.
